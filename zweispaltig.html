HTML und CSS

HTML

HTML steht nicht für "Hast du mich lieb" und auch nicht für "Hier tauchen Mäuse-Liebhaber", sondern für Hypertext Markup Language (englisch für Hypertext Auszeichnungssprache).

HTML ist eine textbasierte Auszeichnungssprache zur Strukturierung von Dokumenten wie Texten mit Verlinkungen, , Bildern, Videos und anderen Inhalten. HTML-Dokumente sind die Grundlage des World Wide Web und werden von Browsern wie Firefox, Chrome, Safari oder Edge dargestellt. HTML-Dateien können außer auch Metainformationen beinhalten, beispielsweise zum benutzten Zeichensatz oder eine Beschreibung des Seiteninhalts.

Früher, also so ungefähr im Mittelalter, hat man noch Layouts über HTML-Tabellen umgesetzt, aber diese dunklen Tage sind glücklicherweise längst vorbei.

CSS

CSS bezeichnet das Computerspiel "Counter Strike" und könnte auch für "Chaos siegt sicher" stehen. In unserem Zusammenhang bedeutet die Abkürzung aber Cascading Style Sheets (englisch für kaskadierende Formatvorlagen).

CSS ist die Formatierungssprache für HTML. Man kann aber beispielsweise auch XML-Dokumente oder SVG-Grafiken damit formatieren. HTML und CSS sind das Dreamteam fürs Web. CSS wird vom World Wide Web Consortium (W3C) beständig weiterentwickelt. Die einzelnen Features von CSS werden in vielen einzelnen Spezifikationen entwickelt, die selbst wieder versioniert werden.

CSS wurde entworfen, um eine Trennung von Inhalten und Formatierungen zu ermöglichen. Mit HTML sagen Sie "das hier ist eine Überschrift" mit CSS bestimmen Sie hingegen, wie diese aussehen soll. Sie können Farben, Schriften, Abstände, Rahmen definieren und mit CSS das Layout festlegen.
